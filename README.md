# CC Books

Превращает логи сессий Claude Code в красивые «книги» с перелистыванием страниц. 3D-полка + интерфейс с перелистыванием.


## Что это?

Работает как скилл [Claude Code](https://claude.ai/code): читает логи сессий за день и автоматически генерирует книгу с перелистыванием страниц прямо в браузере.

- **Перелистывание мышкой** — хватай страницу и тяни. Можно остановиться на полпути. Отпустишь — страница либо перевернётся, либо пружинисто вернётся обратно
- **Звук бумаги** — синтезируется через Web Audio API (никаких аудиофайлов)
- **3D-эффект** — CSS 3D transform, тень от корешка, загиб страницы
- **Работает на телефоне** — поддержка тач-управления
- **Без зависимостей** — один HTML-файл. Без сборки, без библиотек

## Установка (для пользователей Claude Code)

```bash
git clone https://github.com/tipsai/cc-books.git
cp -r cc-books/skill/ ~/.claude/skills/daily-flipbook/
```

В Claude Code:

```
/daily-flipbook
```

Всё. Claude прочитает сегодняшние логи сессий, разобьёт на главы и откроет книгу в браузере.

### Как это работает

Claude Code сканирует логи сессий за сегодня из:
- `~/.claude/projects/` (по умолчанию)
- `~/claude-data/projects/` (если задан `CLAUDE_CONFIG_DIR`)

Каждая сессия становится главой: что ты спрашивал, что было сделано, чему научился. Результат — один HTML-файл: `/tmp/claude/daily-flipbook/YYYY-MM-DD.html`.

```
Логи сессий (JSONL)
  → Claude извлекает задачи и выводы
  → Генерирует главы с блоками кода, цитатами и подсказками
  → Рендерит в виде книги с перелистыванием
  → Открывает в браузере
```

### Автогенерация (книги копятся каждый день)

Можно настроить cron, чтобы книга за день создавалась автоматически в 23:55:

```bash
# macOS (launchd)
cp skill/com.cc-books.daily.plist ~/Library/LaunchAgents/
launchctl load ~/Library/LaunchAgents/com.cc-books.daily.plist

# Linux (cron)
crontab -e
# 55 23 * * * /path/to/cc-books/skill/daily-cron.sh
```

Книга генерируется только в те дни, когда были сессии. Уже сгенерированные дни пропускаются.

## Использование без Claude Code

Просто открой `index.html` в браузере — увидишь демо. Чтобы изменить контент, отредактируй массив `pages`:

```javascript
const pages = [
  {
    front: `<div class="page-title">Название главы</div>
            <div class="page-body"><p>Содержимое лицевой стороны</p></div>`,
    back:  `<div class="page-body"><p>Содержимое оборотной стороны</p></div>`,
    leftContent: `<p>Отображается слева после перелистывания</p>`
  },
];
```

## Управление

| Действие | Результат |
|----------|-----------|
| Тянуть правую половину влево | Следующая страница |
| Тянуть левую половину вправо | Предыдущая страница |
| Отпустить при угле > 50° | Страница перелистнётся |
| Отпустить при угле < 50° | Страница вернётся обратно |
| `←` `→` `Space` | Клавиатурная навигация |
| Кнопки | Кнопки навигации внизу |

## CSS-классы

| Класс | Назначение |
|-------|------------|
| `.page-title` | Заголовок главы (крупный) |
| `.chapter-label` | Метка главы («Глава первая» и т.д.) |
| `.page-body` | Основной текст (`.dropcap` для буквицы) |
| `.code-block` | Блок кода (`.comment` `.keyword` `.string` `.property`) |
| `.quote` | Блок цитаты |
| `.tip-box` | Блок подсказок (`.tip-title` + `<ul>`) |
| `.comparison` | Двухколоночное сравнение (`.col.good` / `.col.bad`) |
| `.divider` | Декоративная разделительная линия |
| `.cover-front` | Дизайн обложки |

## Кастомизация

### Цвета темы

```css
/* Обложка */
.cover-front { background: linear-gradient(145deg, #1a1520, #2a1f30); }

/* Акцентный цвет (метки глав, подсказки, буквицы) */
.chapter-label, .tip-box .tip-title, .page-body .dropcap::first-letter { color: #D97B2F; }

/* Фон страницы */
.page .front { background: linear-gradient(135deg, #fefcf7, #f5f0e8); }
```

### Громкость

```javascript
g1.gain.setValueAtTime(0.4, ...);  // Звук перелистывания (0 — без звука)
g.gain.value = 0.3;                // Шуршание при перетаскивании (0 — без звука)
```
